## Información General

**Aplicación:** CRM con clientes y notas  
**Stack Tecnológico:** HTML + JavaScript  
**Repositorio:** https://github.com/luisernestosalas/app-ugcqn  
**URL de Producción:** https://app-ugcqn-nb3lfqyna-luisernestosalas-2775s-projects.vercel.app  

## Descripción del Proyecto

Sistema de gestión de relaciones con clientes (CRM) desarrollado con tecnologías web básicas, enfocado en la administración de información de clientes y gestión de notas asociadas.

## Arquitectura Técnica

### Frontend
- **HTML:** Estructura y maquetado de la interfaz de usuario
- **JavaScript:** Lógica de negocio, manipulación del DOM e interactividad

### Deployment
- **Plataforma:** Vercel
- **Tipo:** Aplicación web estática

## Funcionalidades Principales

### Gestión de Clientes
- Registro de nuevos clientes
- Visualización de información de clientes
- Edición de datos de clientes
- Eliminación de registros de clientes

### Sistema de Notas
- Creación de notas asociadas a clientes
- Visualización de notas por cliente
- Edición de notas existentes
- Eliminación de notas
