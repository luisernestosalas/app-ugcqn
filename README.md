# CRM con Clientes y Notas

Una aplicación web de gestión de relaciones con clientes (CRM) desarrollada con tecnologías web estándar que permite administrar clientes y sus notas asociadas de manera eficiente.

## 🚀 Demo en Vivo

Accede a la aplicación: [https://app-ugcqn-nb3lfqyna-luisernestosalas-2775s-projects.vercel.app](https://app-ugcqn-nb3lfqyna-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- **Gestión de Clientes**: Crear, editar, visualizar y eliminar clientes
- **Sistema de Notas**: Agregar notas detalladas para cada cliente
- **Interfaz Intuitiva**: Diseño responsive y fácil de usar
- **Almacenamiento Local**: Los datos se guardan en el navegador
- **Búsqueda y Filtrado**: Encuentra clientes rápidamente
- **Sin Backend**: Funciona completamente en el frontend
